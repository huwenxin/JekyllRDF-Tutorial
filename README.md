# Jekyll-RDF Tutorial

## Preparation

- Install [Ruby](https://www.ruby-lang.org/) >= 2.1
- Install [Bundler](https://bundler.io/)

## 1. Add Your Data

In the first step you select the dataset you want to publish. Put it into the `_data` directory.

    $ mkdir _data
    $ cp your_dataset.ttl _data/graph.ttl
